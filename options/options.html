配图助手设置

配置图片搜索 API 密钥，密钥免费申请，用于搜索真实照片

Unsplash API

提供高质量真实摄影图片，免费额度：50 次/小时

  1. 访问 unsplash.com/developers
  2. 注册/登录，创建新应用
  3. 复制 Access Key 粘贴到下方

Pexels API

提供高质量真实摄影图片，免费额度：200 次/小时

  1. 访问 pexels.com/api
  2. 注册/登录，获取 API Key
  3. 复制 API Key 粘贴到下方

OpenRouter AI（关键词提取）

文字超过10字时，自动用AI提取最多5个关键词再搜索，结果更精准。免费模型可用

  1. 访问 openrouter.ai/keys
  2. 注册/登录，创建 API Key
  3. 复制 Key 粘贴到下方（选填，不配则直接用原文搜索）

💡 使用说明

  • 至少配置一个 API 密钥即可使用
  • 两个都配置可获取更多结果
  • 密钥仅存储在本地浏览器，不会上传
  • 搜索后点击图片即可复制图片 URL，粘贴到公众号编辑器即可
