## 基础信息

- **名称 (中文):** 配图助手 — 公众号配图搜索
- **名称 (English):** Image Finder — Real Image Search for Articles
- **版本:** 1.0.0
- **分类:** 照片 / 生产工具 (Photos / Productivity)

## 详细描述

### 中文
配图助手是一款帮助公众号作者、自媒体人快速找到真实配图的浏览器插件。

输入文章段落或关键词，即可从 Unsplash、Pexels 等真实摄影图库中搜索高质量照片，告别 AI 假图。

**核心功能：**
• 输入文字搜索真实照片（非 AI 生成）
• 选中文字右键 → 「搜索配图」，快速查找
• 文字过长时自动提取关键词，结果更精准
• 点击图片自动复制 URL，粘贴到编辑器即可
• 支持中英文界面，自动适配浏览器语言

**使用方式：**
1. 安装后在设置页配置 API 密钥（免费申请）
2. 在公众号编辑器中选中文字，右键 → 「搜索配图」
3. 或点击工具栏图标，手动输入文字搜索
4. 点击任意图片复制 URL，粘贴到文章

**所需 API 密钥（免费）：**
- Unsplash: unsplash.com/developers（50次/小时）
- Pexels: pexels.com/api（200次/小时）
- OpenRouter（可选）: openrouter.ai/keys

### English
Image Finder is a browser extension that helps writers and content creators find real, authentic images for their articles.

Simply enter text or keywords to search high-quality photos from real photography sources like Unsplash and Pexels. No AI-generated images.

**Key Features:**
• Search real photos by text (not AI-generated)
• Right-click selected text → "Search Images" for quick results
• Auto-extract keywords for long text to improve accuracy
• Click image to copy URL, paste directly into your editor
• Supports Chinese and English UI (auto-detects browser language)

**How to Use:**
1. Configure API keys in settings (free to obtain)
2. Select text in your article editor, right-click → "Search Images"
3. Or click the toolbar icon and enter text manually
4. Click any image to copy the URL and paste into your article

**Required API Keys (free):**
- Unsplash: unsplash.com/developers (50 req/hour)
- Pexels: pexels.com/api (200 req/hour)
- OpenRouter (optional): openrouter.ai/keys

## 提交步骤

### Chrome Web Store
1. 访问 https://chrome.google.com/webstore/devconsole
2. 注册开发者账号（$5 一次性费用）
3. 点击 "New item" 上传 ZIP（`配图助手_v1.0.0.zip`）
4. 填写以上描述信息，上传截图
5. 提交审核（通常 1-3 个工作日）

### Edge Add-ons
1. 访问 https://partner.microsoft.com/dashboard/microsoftedge/
2. 使用微软账号登录（免费）
3. 创建新提交，上传相同 ZIP
4. 填写描述信息，上传截图
5. 提交审核（通常 1-2 个工作日）
